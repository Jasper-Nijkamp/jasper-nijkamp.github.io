My Projects

Procedural Planets

In this Unity project I used my prior knowledge in Unity shaders to create a custom ray traced scene. The project supports ray tracing at around 30 fps for both primitive shapes and custom 3D-Models. The goal of this project was to improve my skills in shader programming and optimizing my code for future projects.

The Scorch

In this group project me and 4 classmates created a prototype for a open-world game focused on exploration in 3 weeks. In the game the player uses a boat to sail over the desert sand in search of adventure. The player encounters many enemies while trying to find out what happened to the world.
